충남 당일 코스

최신 UI(Vite / React)는 빌드가 있을 때 이 주소에서 열립니다.

로컬에서 한 번만 실행

  1. 터미널에서 프로젝트의 frontend 폴더로 이동
  2. npm install
  3. npm run build → frontend/dist 가 생깁니다
  4. 이 서버(python main.py)를 다시 시작한 뒤 이 페이지를 새로고침

frontend/dist/index.html 이 있으면 백엔드가 자동으로 Vite UI를 씁니다. (예전에는 이 파일이 없을 때 static 의 긴 HTML이 뜨는 것이 기본이어서, 구 UI와 헷갈릴 수 있었습니다.)
